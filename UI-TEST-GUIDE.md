## 📋 技术改进

### JavaScript 改进
- 创建统一的 `scrollToQuestion()` 函数：用于滚动到题目内容
- 创建统一的 `scrollToTop()` 函数：用于滚动到页面顶部
- 答题导航（Next/Previous/题号点击）：调用 `scrollToQuestion()`
- 页面切换（提交/查看答案/重新开始）：调用 `scrollToTop()`
- 使用 `scrollIntoView()` 和 `window.scrollTo()` API 实现平滑滚动

### CSS 改进
- 添加 `scroll-margin-top` 属性，确保题目不被header遮挡
- 桌面设备：20px 边距
- 移动设备：100px 边距（考虑header高度）

### 用户体验改进
- 无需手动滚动
- 视觉焦点始终在题目内容上
- 平滑的滚动动画
- 响应式设计，适配所有设备
